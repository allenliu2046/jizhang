# 记账

极简个人记账 PWA：打开即输入金额，两秒记完一笔。

- **记账**：自绘数字键盘，输入金额直接「记一笔」，长按 ⌫ 清空
- **统计**：今日 / 本周（周一起）/ 本月汇总，近 7 天柱状图，本月分类占比
- **明细**：按天分组，点任意一笔可补备注、改分类、改金额或删除
- **数据**：只存手机本地（localStorage），支持 JSON 导出 / 导入备份

## 使用

iPhone Safari 打开部署地址 → 分享 → 添加到主屏幕，即可像 App 一样全屏使用，离线可用。

## 开发

纯 HTML/CSS/JS，无构建步骤。本地预览：

```sh
node tools/serve.js 8300
```

图标由 `tools/make_icons.py` 生成（纯 Python 标准库）。

发布新版本时递增 `sw.js` 里的 `CACHE` 版本号，客户端下次联网打开会自动更新。
