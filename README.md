# 记账

**一个两秒钟记完一笔的极简记账应用。**

打开就是数字键盘，按下金额、点「记一笔」，完成。免费、无广告、无注册，账本只存在你自己的手机里。

## 三步装到手机上

1. 用 **Safari**（iPhone）或 **Chrome**（安卓）打开上面的应用链接
2. iPhone 点**分享按钮**、安卓点**浏览器菜单**，选**「添加到主屏幕」**
3. 以后从主屏幕的绿色 ¥ 图标打开——全屏运行、离线可用，和普通 App 没有区别

## 为什么它长这样

**记账最大的敌人不是数学，是放弃。**

几乎每个坚持不下来的账本，都是同一个剧本：付钱的那一刻正忙，而记账软件要你选账户、挑分类、打标签、写备注——嫌麻烦，想着「回头再记」；到了晚上，要么忘得干干净净，要么对着支付记录苦苦回忆。回忆是要消耗精力的，而且回忆出来的账并不可靠：靠回忆补记，最容易漏掉的恰恰是小额、高频的日常消费——最该被看见的那部分。又累，又不准，几天之后，人自然就放弃了。

所以这个应用把宝全押在一件事上：**让「记下来」这个动作便宜到无法拒绝。**

### 两秒钟法则

打开应用，数字键盘已经等在那里；按下金额，点「记一笔」——两秒，完成。人的动机每天都在波动，靠不住；唯一稳定的办法，是把事情本身变容易。对一天要发生好几次的动作，每省掉一步点击，它活下来的概率就大一分。

### 「回头再记」是个骗局

「记得稍后去做某事」是人类记忆里最不可靠的功能之一——心理学称之为前瞻记忆，没有外部提醒时失败率极高。这不是懒，是人脑的出厂设置。与其对抗天性，不如顺应它：在消费发生的当下、记忆还热着的时候，用两秒钟把数字放进账本。

### 先记下来，想整理的时候再整理

付钱的那一刻只需要记一个数字。备注和分类都可以之后在明细里随手补——也可以永远不补，应用照样把每天、每周、每月的数字算得清清楚楚。**捕捉要快，整理要闲**，它们本就该发生在不同时刻。多数记账软件把所有字段堆在录入那一刻，等于让每一笔账都交一遍「填表税」。

### 没有的功能，是故意没有的

没有预算提醒，没有账户体系，没有云同步，没有签到和小红点。每一个候选功能都要过同一道审判：**它会不会拖慢那两秒？** 会，就不做。

### 它不打扰你，但立刻回应你

从不推送、从不提醒——工具应该在被召唤时出现，其余时间保持沉默。但每记一笔，「今日已花」立刻跳一下：这一眼就是坚持下去的燃料。

### 一个诚实的提醒

零摩擦解决的是「嫌麻烦」，解决不了「忘了打开」。头两三个星期，请有意识地把「付完钱」和「顺手记」绑在一起——因为动作只要两秒，这条反射弧很快就会焊牢。

### 时间会给你回报

今天随手按下的一个数字，一个月后是月度汇总里的一行，一年后是十二行。记录越久，这本账越值钱。

## 功能一览

- **记账**：自绘数字键盘，输入金额直接「记一笔」，长按 ⌫ 清空
- **统计**：今日 / 本周（周一起）/ 本月卡片、累计总额、近 7 天柱状图、月度汇总（可回看任意历史月份总额）、本月分类占比
- **明细**：按天分组，点任意一笔可补备注、选分类、改金额、删除（两次确认）
- **数据**：只存手机本地（localStorage），JSON 一键导出 / 导入，换机无忧
